# random-word-giphy
Learn promises by building app that fetches an image when given a random word

**To setup app:**
1. Get API key from https://developers.giphy.com/
2. Create `.env` file and add key: GIPHY_API_KEY=addYourKeyHere

**To start app**
- `yarn` 
- `yarn start`

**Skills:**
- JavaScript Promises
